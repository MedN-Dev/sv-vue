# Router Design

## 多項目

| URL | Descript | 
| ------- | -------- |
| `/` | 多項目下選中所有分类 (默认) | 
| `/summary/all` | 多項目下選中ALL分类 |
| `/summary/rj` | 多項目下選中RJ分类 |

## 单项目

| URL | Descript | 
| ------- | -------- |
| `/project/:id` | 单項目下summary总览 (默认) | 
| `/project/:id/energy` | 单項目下energy | 
| `/project/:id/event` | 单項目下event | 
| `/project/:id/detail` | 单項目下detail |
